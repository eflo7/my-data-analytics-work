## Projects

### 1. [Alteryx Health Spatial Analysis](./alteryx-health-spatial-analysis)

This project looks at the relationship between health outcomes and restaurant density across different geographic areas.

**What I worked on:**

* Built an Alteryx workflow to combine health, restaurant, and location-based data
* Cleaned and prepared messy fields for analysis
* Used spatial tools to compare restaurant locations with health-related patterns
* Organized the project so the workflow and results could be reviewed clearly

**Main tools:** Alteryx, spatial analysis, data cleaning, workflow design

### 2. [Restaurant Database Design](./restaurant-database-design)

This project focused on creating a relational database for a restaurant-style business case.

**What I worked on:**

* Designed a database structure for sales, employees, menu items, and inventory
* Created relationships between tables to reduce repeated data
* Built queries to answer business questions
* Created forms and reports to make the database easier to use

**Main tools:** Microsoft Access, SQL, ER diagrams, database design

### 3. [NFL Draft Attendance Analysis](./nfl-draft-attendance-analysis)

This project analyzes NFL Draft attendance trends and how major sports events can create value for cities, leagues, and fans.

**What I worked on:**

* Organized attendance data across NFL Draft host cities
* Compared event performance over time
* Looked for patterns connected to location, fan interest, and event growth
* Built a project summary that connects the data to sports business strategy

**Main tools:** Excel, data analysis, sports business research, visualization

### 4. [Major Pool 2026 Golf Analysis](./major-pool-2026-golf-analysis)

This project was built around a golf pool strategy model for the 2026 major season.

**What I worked on:**

* Organized golfer data, picks, and scoring information
* Compared player options using a structured rating system
* Created files to help make more strategic pool decisions
* Used AI-assisted workflows to help structure the model, clean the information, and explain the strategy

**Main tools:** Excel, sports analytics, scoring models, AI-assisted analysis
